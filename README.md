# transmit-file

### 使用nextjs框架实现的在线文件传输，部署在[vercel](https://vercel.com/)上

#### 如何使用

在浏览器环境访问[网站](https://a.952737.xyz/)，一端选择发送，另一端选择接收，接收端需要填入发送端生成的一个会话代码。操作简单
